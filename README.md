# ITP4506-GUI
This is a complete GUI-based Food Ordering and Restaurant Management System built with HTML, CSS (Bootstrap), and JavaScript (jQuery). 

# For Customers:

Browse restaurant menu with filtering and search options

Add items to cart with customizable options

Real-time price calculation

Order tracking with visual timeline

Multiple payment methods (credit card, cash, digital wallets)

Favorite items and address management

# For Restaurant Staff:

Menu Management - Add/edit food items with images, pricing, and preparation times

Order Processing - Track in-progress and ready-for-pickup orders

Inventory Control - Set item status (Available/Unavailable/Sold Out)

Customer Communication - Built-in chat system for order queries

Order Analytics - View all orders with detailed status tracking

# Technical Features:

Responsive design that works on all devices

Interactive modals for food customization

Drag & drop image upload for menu items

DataTables for efficient order management

Real-time updates and form validation

The system offers an intuitive graphical interface for both customers to place orders and restaurant staff to manage daily operations efficiently.
